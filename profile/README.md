# Curnova

Plataforma modular para que cualquiera pueda crear su propio espacio de formación y enseñanza.

Descripción
-----------
Curnova permite a individuos, organizaciones y comunidades lanzar rápidamente una plataforma educativa personalizada. Ofrece herramientas para crear cursos, gestionar alumnos, evaluar progreso y publicar contenidos —sin necesidad de experiencia técnica—.

Por qué Curnova
----------------
- Flexible: adapta la plataforma a distintos modelos educativos (cursos, bootcamps, talleres, mentorías).
- Fácil de usar: interfaz pensada para instructores y administradores no técnicos.
- Modular: integra contenidos en video, texto, cuestionarios y proyectos prácticos.
- Escalable: prepara tu comunidad para crecer sin perder control sobre contenidos y permisos.

Características principales
-------------------------
- Creación de cursos y rutas de aprendizaje.
- Gestión de estudiantes y grupos.
- Evaluaciones automáticas y rúbricas para tareas prácticas.
- Panel de analíticas para seguimiento de progreso y engagement.
- Integración con pagos, certificados y acceso restringido.
- Plantillas y temas personalizables.

Casos de uso
-----------
- Academias y escuelas interesadas en digitalizar cursos presenciales.
- Empresas que entrenan equipos internos o partners.
- Formadores independientes que quieren su propia marca y control de contenidos.
- Comunidades que desean ofrecer rutas de aprendizaje estructuradas.

Cómo empezar (rápido)
---------------------
1. Clona o instala la solución base de Curnova.
2. Crea tu primer curso desde el panel de administración.
3. Añade módulos: lecciones en video, lecturas y evaluaciones.
4. Invita a tus primeros estudiantes y prueba una sesión piloto.

Buenas prácticas
----------------
- Define objetivos de aprendizaje claros para cada curso.
- Empieza con contenido mínimo viable y mejora iterativamente.
- Usa evaluaciones prácticas y proyectos para medir impacto real.
- Mantén comunicación frecuente con tus estudiantes para aumentar retención.
